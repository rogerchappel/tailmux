# Roadmap

## v0.1 MVP

- Fixture-backed Tailscale, SSH, and port parsing.
- Dry-run tmux workspace planning.
- Explicit `--live` and `--execute` consent flags.

## Later

- More port discovery adapters (`lsof`, `ss`, `netstat`) with fixture tests.
- Optional health checks per template pane.
- Rich terminal status board.
- Template composition for project-specific workspaces.

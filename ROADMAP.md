# Roadmap

## V1 MVP

- Fixture-backed local status board.
- Shareable tmux session template generation.
- Dry-run launch command planning.
- Safety-first documentation and validation.

## Next

- Optional live `tailscale status --json` import behind an explicit flag.
- Optional tmux template file export.
- Richer service health import formats.
- Better grouping by tags, projects, and node roles.
- Shell completion and package publishing prep.
